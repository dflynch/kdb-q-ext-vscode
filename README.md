# kdb-q-ext-vscode

A lightweight extension for kdb+/q programming.

## Features

- Syntax highlighting
- Run multi-line snippets from \*.q files in the active terminal

## Commands

| Command                   | Keybinding       | When                |
| ------------------------- | ---------------- | ------------------- |
| kdb+/q: Run Selected Text | Ctrl/Cmd + Enter | editorLangId == 'q' |
